Mimpi Saya

02

Entrepreneur

Saya tidak hanya bermimpi mejadi pemimpin perusahaan saya sendiri - saya sudah mulai melangkah. saya pernah membuat produk sendiri sebagai bagian dari perjalanan saya di dunia bisnis. Dari proses ide, desain, produksi, hingga pemasaran, saya belajar banyak hal tentang bagaimana membangun sesuatu dari Nol. pengalaman ini memperkuat tekad saya untuk terus berkembang sebagai seorang Entrepreneur muda.

Produk Pertama
